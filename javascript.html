Operators

Javascript operators are symbols used to perform operations on operands. There are different types of operators in js namely arthimatic operators, assignment operators, comparison operators, string operators, logical operators, bitwise operators.

Arthimatic Operators

A typical arithmetic operation takes two numbers and produces a new number.
addition(+), subtration(-), division(/), multiplication(*), exponentiation(**), Modulus(%), Increment(++), decreement(--) all comes under this arthematic operators.

Assignment operators

Assignment operators assign values to JavaScript variables.

The = Operator - Simple Assignment Operator assigns a value to a variable.
The += Operator - Addition Assignment Operator adds a value to a variable.
The -= Operator - Subtraction Assignment Operator subtracts a value from a variable.
The *= Operator - Multiplication Assignment Operator multiplies a variable.
The **= Operator - Exponentiation Assignment Operator raises a variable to the power of the operand.
The /= Operator - Division Assignment Operator divides a variable.
The %= Operator - Remainder Assignment Operator assigns a remainder to a variable.

Comparison operators

comparison operators are used to compare the values of javascript variables

The == Operator - compares only value of variable that was 'equal to'
The === operator - compares both 'equal value and equal type'
The != operator - compares "not equal"
The !== operator - compares'not equal value or not equal type'
The > operator - compares 'greater than'
The < operator - compares 'lesser than'
The >= operator - compares ' greater than or equal to'
The <= operator - compares ' lesser than or equal to'

String Methods

JavaScript strings are the sequence of characters. In JavaScript, strings automatically convert the string to objects so that we can use string methods and properties also for primitive strings. It is used to represent and manipulate the sequence of characters.

String length - The length property returns the length of a string
String slice() - extracts a part of a string and returns the extracted part in a new string. The method takes 2 parameters: start position, and end position
String substring() - substring() is similar to slice(). The difference is that start and end values less than 0 are treated as 0 in substring()
String substr() - substr() is similar to slice(). The difference is that the second parameter specifies the length of the extracted part.
String replace() - replaces a specified value with another value in a string
String repalceAll() - allows you to specify a regular expression instead of a string to be replaced.
String toUpperCase() - string is converted to upper case
String toLowerCase() - string is converted to lower case
String concat() - joins two or more strings
String trim() - removes whitespace from both sides of a string
